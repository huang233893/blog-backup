## 版本信息

系统版本：Windows 11 25H2

内核版本：26220.7262

频道：Dev Channel、Beta Channel

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/1-8.jpg)

> 微软罕见地在周一面向 Dev 频道与 Beta 频道同步推送了 Windows 11 预览体验版更新，此次更新基于 Windows 11 25H2 版本，核心带来实验性 AI 智能体功能、辅助功能语音升级，同时修复多项稳定性问题，并开放两频道临时切换窗口。


## 核心更新亮点

### 1. 实验性 AI 智能体功能上线

设置应用新增“实验性智能代理功能”开关，位于“系统 > 人工智能组件”（System → AI Components）路径下。该功能开启后将创建独立的“智能体工作区”，类似 Windows 沙箱但更高效，智能体可在隔离运行环境中执行指定任务。

- 权限控制：默认关闭，需管理员权限启用，启用后对所有设备用户生效；
- 数据访问：可配置智能体对桌面、文档、下载等“已知文件夹”的读写权限，所有操作均有可追溯日志；
- 功能状态：当前仅开放开关测试，智能体具体任务执行能力仍在开发中，后续将通过受控推送逐步解锁。


### 2. 辅助功能高清语音与数学朗读升级

旁白（Narrator）和放大镜（Magnifier）正式支持英语（美国）高清语音，基于 Azure 最新端侧文字转语音模型，通过生成式 AI 优化语调和节奏，朗读更自然流畅，可减少长时间聆听疲劳。

- 语音选项：提供 Andrew（男性）和 Ava（女性）两款高清语音，支持调整语速和切换语音；
- 启用路径：设置 > 辅助功能 > 旁白/放大镜，点击“添加自然语音”下载后即可使用；
- 新增能力：旁白首次支持数学内容朗读，初期可结构化朗读 Microsoft 365（如 Word）中的方程式，还原数学表达逻辑，后续将扩展至更多应用场景。


### 3. Click to Do 与教学提示优化

- Click to Do 测试新版操作菜单，集中呈现复制、保存、分享等常用选项，当画面出现大型图像或表格时自动弹出，提升操作便捷性；
- 教程模式新增“启动教程”按钮，优化说明文字表述，帮助用户快速启动并理解教程流程。


### 4. 跨频道切换窗口期开放

Dev 频道用户可临时切换至 Beta 频道，切换窗口仅在两频道同步推送 25H2 系列更新期间开放。一旦 Dev 频道推送更高版本号更新，切换通道将关闭，下次开放可能需等待一年，微软建议用户在窗口期内评估适合的频道归属。


## 功能调整与临时变更

1. 暂时关闭“图像对象选择”功能，具体恢复时间将通过官方公告通知；
2. 重新配置文件资源管理器中 AI 操作（AI Actions）的受控功能推送（CFR）机制，部分用户可能暂时无法使用图像操作和 Copilot 摘要功能，将在配置完成后逐步恢复；
3. 暂停“新文件夹在标签页而非新窗口打开”功能推送，因前期发现功能异常需优化完善；
4. 暂时停止“管理员保护”功能推送，后续将基于用户反馈优化后重新上线。


## 修复与稳定性优化

1. 修复 WinRE 与高级启动环境中部分用户键鼠失效的问题，确保应急启动场景的操作可用性；
2. 解决任务管理器关闭后进程未正确停止，以及启动时意外弹出的问题；
3. 修复设置 > 系统 > 高级中虚拟工作区选项失效的问题，现已重新开放使用；
4. 修复设置 > 系统 > 关于中 CPU 信息意外截断的显示问题。


## 已知问题

1. 部分设备点击开始菜单无法打开，可通过 Windows 键正常呼出，该问题可能同时影响通知中心（可通过 Win+N 打开）；
2. 部分用户的系统托盘未显示应有的应用图标，需等待后续修复补丁；
3. 深色模式下的复制对话框，文本缩放时滚动条和页脚会消失，替换为白色块显示异常；
4. ARM64 设备上的预览体验成员可能遇到 Visual Studio 或依赖 .NET Framework 的应用崩溃，安装最新 .NET Framework 更新可解决此问题。


## 更新提示

1. 推送机制：更新采用受控功能推送（CFR）技术，新功能将先向部分用户推送，再根据反馈逐步扩大范围；开启“设置 > Windows 更新 > 获取最新可用更新”开关可优先体验新功能；
2. 安装方式：符合频道资格的用户可在“设置 > Windows 更新”中手动检查并安装，更新过程中需保持网络稳定，避免中断；
3. 风险提示：该版本为预览开发版，稳定性不及正式版，不建议在生产设备、关键工作设备或无数据备份的设备上安装；部分功能尚未完全本地化，后续将逐步完善；
4. 反馈渠道：使用中遇到问题可通过 Win+F 打开“反馈中心”，在对应分类下提交详细场景，助力功能优化。
