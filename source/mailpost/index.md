## 欢迎订阅酥米的小站邮箱推送

由于本站为静态博客，只能使用**LiuShen**大佬的 `Friend-Circle-Lite` 插件进行邮箱推送功能，在此谢谢大佬的贡献！

### 可以通过以下方式进行邮箱订阅本站

**初次进入github进行邮箱订阅可能需要科学或者加速器，大部分时间不需要**

1. 前往邮箱订阅网站

{% link 邮箱订阅处,酥米的小站,https://github.com/huang233893/Friend-Circle-Lite/issues %}

2. 选择“New issues”

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm9.jpg)

3. 选择”邮箱订阅“

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm8.jpg)

4. 按照要求填写邮箱至标题后面

**请在标题后直接输入您的邮箱，不要带空格及其他符号**

- 输入邮箱前
![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm7.jpg)

- 输入邮箱后
![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm5.jpg)

5. 点击”Create“

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm6.jpg)

6. 显示回复即完成

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm4.jpg)

### 取消订阅

1. 前往邮箱订阅网站

{% link 邮箱订阅处,酥米的小站,https://github.com/huang233893/Friend-Circle-Lite/issues %}

2. 选择“closed”标签

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm3.jpg)

3. 选择你的邮箱账户

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm1.jpg)

4. 往下划，找到右下角“Delete issue”，点击它即可取消订阅

![](https://cdn.sumi233.top/gh/huang233893/blog-image-bed@main/top/huang233893/imgs/blog/postm2.jpg)

如果遇到问题，请在下方评论区留言告诉酥米哦~
